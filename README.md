# 🏆 Gold Price Prediction Using Machine Learning

## 📌 Project Overview

This project predicts **gold prices (GLD)** using the **Random Forest Regression** algorithm. It demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature selection, model training, prediction, and performance evaluation.

The objective is to build an accurate regression model capable of predicting gold prices based on various financial indicators.

---

## 🎯 Objectives

* Understand the Gold Price dataset.
* Perform Exploratory Data Analysis (EDA).
* Analyze feature correlations.
* Build a Random Forest Regression model.
* Predict gold prices accurately.
* Evaluate the model using multiple performance metrics.

---

## 📂 Dataset

The dataset contains historical gold price data along with related financial indicators.

### Features

* **Date** – Trading date
* **SPX** – S&P 500 Index
* **USO** – United States Oil Fund
* **SLV** – Silver Price ETF
* **EUR/USD** – Euro to US Dollar Exchange Rate
* **GLD** – Gold Price (Target Variable)

## 🤖 Machine Learning Model

**Algorithm Used:**

* Random Forest Regressor

### Why Random Forest?

* Handles non-linear relationships effectively.
* Reduces overfitting by combining multiple decision trees.
* Provides high prediction accuracy.
* Works well on regression problems.
